NAME

Google::Ads::AdWords::AuthTokenHandler

Back to Top

DESCRIPTION

A concrete implementation of the Google::Ads::Common::AuthTokenHandler manpage that defines the auth service name required to access AdWords API servers using ClientLogin, see https://developers.google.com/accounts/docs/AuthForInstalledApps for details of the protocol.

Refer to the base object the Google::Ads::Common::AuthTokenHandler manpage for a complete documentation of all the methods supported by this handler class.

Back to Top

METHODS

_service

Method defined by the Google::Ads::AdWords::AuthTokenHandler manpage and implemented in this class as a requirement for the ClientLogin protocol.

Returns

Returns the AdWords API service name used to generate the AuthToken.

prepare_request

Method defined by Google::Ads::AdWords::AuthTokenHandler and implemented in this class as a requirement for sending requests to the AdWords API.

Returns

A new HTTP:Request with the authToken header properly set with the authorization information.
