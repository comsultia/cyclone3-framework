NAME

Google::Ads::Common::CaptchaRequiredError

Back to Top

DESCRIPTION

Captures Captcha authorization required error information.

Back to Top

ATTRIBUTES

Each of these attributes can be set via Google::Ads::Common::CaptchaRequiredError->new(). Alternatively, there is a get_ and set_ method associated with each attribute for retrieving or setting them dynamically.

token

Captcha challenge token.

image

Holds the URL to the captcha image.

url

Holds the URL to unlock the challenge.
