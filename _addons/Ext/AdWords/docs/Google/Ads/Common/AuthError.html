NAME

Google::Ads::Common::AuthError

Back to Top

DESCRIPTION

Captures authorization error information.

Back to Top

ATTRIBUTES

Each of these attributes can be set via Google::Ads::Common::AuthError->new(). Alternatively, there is a get_ and set_ method associated with each attribute for retrieving or setting them dynamically.

message

Holds the error message.

code

Holds the HTTP code associated to the error.

content

Holds the body of the error response.
